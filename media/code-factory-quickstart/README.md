# Code Factory 60-second quick start

The composition uses the actual 1920x1080 Factory Studio capture in
`public/factory-studio-control-room-1080.png`. Scene focus regions map to the
exact shipped panel geometry; no substitute dashboard cards are drawn.

Render:

```powershell
npm install
npm run render
```

The output is `output/marketing/code-factory-quickstart-v016.mp4`.
